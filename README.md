## Что это

**Escape** — сессионная мини-игра «побег из тюрьмы»: до 12 заключённых, одна арена,
20 минут беспредела и ровно **один** выживший. Никаких команд, никаких друзей —
только ты, золото и решётки, которые жалобно гнутся под твоей Вилкой.

> *«Игрок споткнулся»* · *«Игрок не справился с управлением»* · *«Игрок выбрал лёгкий путь»*
> — некрологи, которые ты будешь читать чаще, чем хотелось бы.

## Как это играется

1. **Лобби.** Пока набирается народ — «Избиение в лобби»: урон не проходит, но
   считается. Перед стартом объявляют топ-3 самых буйных. Разомнись.
2. **Высадка.** Каждому: золотая кирка **«Вилка...»** *(нелегально... поможет
   сбежать...)*, **24 золота** и **«Личный проводник»** — бумажка, которая знает
   карту лучше тебя.
3. **Гринд.** Лутай сундуки (они пополняются!), выполняй **контракты** — от
   «сломай 10 решёток» до «убей двоих», торгуйся с **жителями-барыгами**,
   копай руду... только осторожно: каждая пятая жила — **чешуйницы**.
4. **Хаос.** Каждые 3,5 минуты — случайное событие: «встань под крышу или молния»,
   «присядь или полтора сердца», «сними всю броню — заплатим». Раз в 10 минут —
   зарплата с неба, буквально с молнией.
5. **Час расплаты.** За 10 минут до конца **все светятся** — прятки окончены.
   На нуле таймера — телепорт на **финальную битву**. Последний живой забирает всё.

Валюта — золотые слитки **прямо в инвентаре**. Умер — уронил. Убил — подобрал.
Экономика честная, как тюремный обмен.

## Фичи

| | |
|---|---|
| 📜 **Контракты** | 6 типов заданий (KILLS / ACTIVATE / MINE / FIND / BREAK / LOOT), прогресс живёт **в самой бумаге** — контракт можно выбить из владельца вместе с прогрессом |
| 🧑‍🌾 **Торговцы** | Жители-барыги с настраиваемыми магазинами за золото |
| 🗺️ **Личный проводник** | 4 подсказки на кулдаунах: ближайший сундук, случайная жертва, ближайший барыга, координаты мест |
| ⚡ **Случайные события** | 5 штук, все с честными проверками (в отличие от оригинала — см. ниже) |
| 📦 **Живой лут** | Взвешенный пул предметов per-arena, автопополнение пустых сундуков через 3 минуты |
| 🏰 **Мультиарена** | Сколько угодно арен, каждая — папка с YML-конфигом; настройка маркерами-предметами прямо в мире |
| 🧪 **Столы зачарований** | Уровни бесплатны (XP-шкала занята таймером) — платишь лазуритом |
| 📊 **Статистика** | SQLite: победы, убийства, MVP, рекорды — `/escape stats` |
| 🔄 **Нулевой след** | Полный снапшот игрока до матча и полный откат мира после: решётки, руды, сундуки — всё как было |

## Быстрый старт

```bash
./gradlew build     # jar → build/libs/EscapeReborn-1.0.0.jar (JDK 25 скачается сам)
./gradlew deploy    # сборка + деплой на тестовый сервер
```

Дальше — настройка арены **без единого конфига руками**: команды выдают
маркеры-предметы, ставишь блок — точка записана. Полная цепочка (арена →
торговцы → контракты → лут → enable) — в
[docs/08-reborn-guide.md](docs/08-reborn-guide.md).

```
/escape                  меню арен          /escape stats [ник]   статистика
/escape join [ID]        в бой              /escape info          правила
/escape leave            сдаться            /escape help          всё остальное
```

## Технологии

**Paper 26.1.2** · **Java 25** · **Gradle 9.6.1** (Kotlin DSL, wrapper в репо) ·
**Adventure + MiniMessage** (все тексты в [`messages.yml`](src/main/resources/messages.yml),
ни одной строки в коде) · **SQLite** (встроенный в Paper) · **PDC** для всех данных
на предметах — никакого парсинга лора · ноль внешних зависимостей, ноль NMS.
