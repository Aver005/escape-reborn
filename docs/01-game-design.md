# Концепция и игровой цикл

## Концепция

- **Жанр:** сессионный last man standing (все против всех) с элементами лут-геймплея, квестов и торговли.
- **Сеттинг:** побег из тюрьмы. Атрибуты: золотая кирка называется «Вилка...» с лором «Нелегально... Поможет сбежать... Сломайте железную решётку...», на карте железные решётки, которые можно выламывать, скрытные подсказки «проводника».
- **Формат:** мультиарена — на сервере может быть сколько угодно арен, каждая привязана к своему миру. Матчи идут независимо.
- **Игроки:** по умолчанию 2–12 на арену (настраивается: min 1–52, max 2–128 при создании).
- **Длительность матча:** 30 минут + финальная битва до последнего выжившего.

## Фазы матча

```
Меню выбора арены → Лобби (разминка) → Отсчёт 20 сек → Генерация арены →
Матч 30 минут (события, лут, контракты, PvP) → Финальная битва → Победа → Очистка
```

### 1. Вход в игру

- `/escape` (или `/escape join` без аргумента) открывает GUI **«Escape»** (54 слота, рамка из чёрных стеклянных панелей).
- Каждая включённая и не начавшаяся арена — блок **лаймовой шерсти**: название арены, лор «Игроков: X / Y», описание, «Нажмите, чтобы войти».
- Начавшиеся арены в меню не показываются, выключенные (`disable`) — тоже.
- Альтернатива: `/escape join <ID арены>` напрямую.
- Если арена заполнена — «Данная арена заполнена. Попробуйте другую :(».

### 2. Лобби

При входе в лобби:

- Полное состояние игрока (инвентарь, броня, позиция, HP, еда, XP, гейммод, эффекты, полёт, скорости, displayName) **сохраняется на диск** и восстановится после матча.
- Игрока очищают: Adventure-режим, 20 HP, 20 еды, **уровень 60** (декоративное значение), телепорт на точку лобби.
- В 8-й слот выдаётся **магма-крем «Выйти»** («Нажмите ПКМ, чтобы выйти») — ПКМ с ним в руке выводит из матча.
- Голод в лобби не тратится, дроп предметов запрещён.
- Чат лобби изолирован: префикс `[Лобби]`, сообщения о входе/выходе («Игрок X зашёл на арену Y»).

**Разминка «Избиение в лобби»:** игроки могут бить друг друга. Урон **не наносится** (ставится в 0), но суммируется в таблицу. За 5 секунд до старта в чат выводится рамка:

```
=============================
  Избиение в лобби:
    Победитель <ник>
    Урон: <N> сердец
=============================
```

### 3. Отсчёт старта

- Начинается автоматически, когда игроков в лобби становится ровно `minPlayers` (по умолчанию 2). Отсчёт — **20 секунд**.
- Каждые 10 секунд и последние 10 — сообщение в чат «Начало игры через N секунд...»; каждые 10 секунд и последние 5 — тайтл «**Начало матча** / через N сек.».
- Уровень XP каждого игрока показывает секунды до старта.
- Если кто-то вышел и игроков стало меньше минимума — отсчёт отменяется.

### 4. Генерация арены на старте

При достижении нуля:

1. **Сундуки** — из всех сконфигурированных точек случайно выбирается `chestCount` (по умолчанию 75), там ставятся сундуки и наполняются лутом (см. механики). Под каждым — невидимая стойка для брони (служебный маркер пустоты/рефилла + кликабельна для открытия сундука).
2. **Торговцы** — из всех точек жителей случайно выбирается `villagersCount` (по умолчанию 32). Житель: без AI, неуязвим, с именем типа торговца над головой.
3. **Руды** — на каждой рудной точке ставится случайный блок: уголь (5/19), железо (4/19), золото (3/19), алмаз (2/19), изумруд (1/19) или **заражённый камень с чешуйницей** (4/19 ≈ 21% — ловушка).
4. **Игроки** — телепорт на случайные (уникальные) точки спавна, Survival, полные HP/еда, уровень 0. Стартовые эффекты: **Сопротивление II — 20 сек, Скорость I — 12 сек, Регенерация I — 20 сек** (защита от спавн-раша).
5. **Стартовый набор** (инвентарь очищен):
   - **«Вилка...»** — золотая кирка. Лор: «Нелегально... / Поможет сбежать... / *Сломайте железную решётку...*»
   - **24 золотых слитка** — стартовый капитал.
   - **«Личный проводник»** — бумага. Лор: «Помогает ориентироваться по карте / Даёт подсказки БЕСПЛАТНО / Для использования ПКМ».

### 5. Основной матч (30 минут)

- Уровень XP каждого игрока = оставшиеся секунды матча (обратный отсчёт с 1800).
- Игрок в чате арены — канал `[Всем]`.
- Каждые **210 секунд** — случайное событие (5 видов, подробно в механиках).
- Каждые **600 секунд** — на каждом игроке визуальная молния и **+16 золота** («зарплата»).
- Оповещения «До финальной битвы N минут...» каждые 10 минут и посекундно за 15 секунд до конца.
- Деятельность игрока: лутать сундуки, выполнять контракты, покупать снаряжение у торговцев, копать руды для контрактов, PvP, ломать решётки для перемещения по карте.

### 6. Эндшпиль

- На отметке **10 минут до конца**: все игроки получают **перманентное свечение** (Glowing) + **18 золота**, в чат: «Все игроки подсвечены.» — прятки закончились.
- На **нуле таймера**: «Переносим на арену финальной битвы...» — всех телепортируют обратно на точки спавна. Дальше матч идёт без таймера до последнего выжившего.

### 7. Смерть

Смерть перехватывается **до** реальной гибели (при летальном уроне):

- Урон отменяется, все вещи высыпаются на землю (дропы запоминаются и убираются после матча), HP восстанавливается, игрок переводится в **Spectator**.
- В чат — случайное шуточное сообщение: «Игрок X **не справился с управлением**» / «**споткнулся**» / «**выбрал лёгкий путь**», затем «Осталось N игроков».
- Погибший попадает в отдельный чат `[Наблюдателям]` и наблюдает до конца матча.
- Выход с сервера во время матча = смерть (вещи выпадают на месте выхода).

### 8. Победа и завершение

Когда остаётся один игрок:

```
=========================================================
[ПОБЕДА] Победителем становится <ник>!
[ПОБЕДА] Квестов выполнено: <N>
[ПОБЕДА] Убито: <N> игроков
[ПОБЕДА] Трейдов сделано: <N>
=========================================================
```

- Победителю включается полёт (если разрешён), запускается отсчёт конца `Stop(10)` — 10 секунд.
- Если игроков не осталось вовсе — `Stop(3)`.
- **Очистка арены:** сундуки убираются, стойки и жители удаляются, сломанные решётки восстанавливаются, дропы зачищаются, все таймеры рефилла отменяются.
- **Восстановление игроков:** каждому возвращается сохранённое состояние (инвентарь, позиция, эффекты и т.д.), флаги и привязка к арене сбрасываются.

## Каналы чата

| Канал | Префикс | Кто видит |
|---|---|---|
| Лобби | `[Лобби] <ник>: ` | Ожидающие в лобби этой арены |
| Игра | `[Всем] <ник>: ` | Живые игроки арены |
| Наблюдатели | `[Наблюдателям] <ник>: ` | Погибшие на этой арене |

Обычный серверный чат для участников матча полностью перекрыт. Команды в матче блокируются — разрешены только `*leave`, `es stop*`, `*start`.
