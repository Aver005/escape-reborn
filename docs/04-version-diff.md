# Сравнение версий 2020 → 2021 → 2025

## Сводка

| | 2020 | 2021 | 2025 |
|---|---|---|---|
| plugin.yml version | `2.0` | `0.5` | `0.5` |
| Ядро | Spigot 1.12.2 | Spigot 1.16.4 (`api-version: 1.16`) | копия 2021 |
| Пакет | `kiviuly.escape` | `kiviuly.Escape` | `kiviuly.Escape` |
| Алиасы | `esc`, `es` | `ecs` (опечатка), `es` | `ecs`, `es` |
| Классы | 9 (те же) | 9 (те же) | 9 (те же) |

**2025 = байт-в-байт копия 2021** (совпадают даже git pack-файлы). Порт на 1.21.5 не был начат — это просто переименованная папка-заготовка.

## Что появилось в 2021 (главные фичи)

1. **Случайные события** — вся система из 5 событий с проверкой через 8 секунд. В 2020 её не было вовсе.
2. **«Личный проводник» / «Личный помощник»** — предмет, GUI и все 4 способности с кулдаунами. В 2020 стартовый набор — только вилка и 24 золота.
3. **Стартовые эффекты** (Сопротивление II 20с, Скорость 12с, Регенерация 20с) — в 2020 игроки высаживались «голыми».
4. **Случайные уникальные спавны** — в 2020 игроков расставляли по точкам последовательно (по порядку списка, предсказуемо).
5. **Тайтлы отсчёта** («Начало матча / через N сек.») — в 2020 только чат.
6. **Открытие сундука кликом по стойке** (armor stand) — новый QoL-обработчик.
7. **`/escape enable|disable <ID>`** — включение/выключение арен.
8. **Tab-completion** — класс Commands стал `TabCompleter`.
9. **Телепорт на финальную битву**: в 2020 — на `time == 1` и по индексу списка; в 2021 — на `time == 0` и на случайные точки.

## Изменения контента и баланса

| Аспект | 2020 | 2021 |
|---|---|---|
| «Вилка...» | Золотая кирка **с прочностью 31/32 — фактически одноразовая**. Лор: «Прокопай путь к свободе... / Сломай железную дверь...» | Полная прочность (32 использования). Лор: «Нелегально... / Поможет сбежать... / Сломайте железную решётку...» |
| Итоги разминки в лобби | Полная таблица урона всех участников (с багом: строки печатали одно и то же значение из-за переприсваивания переменных) | Только победитель и его урон |
| Префикс чата лобби | `[Ожидание...]` | `[Лобби]` |
| Кнопка выхода | «Выход» | «Выйти» |
| Условие продолжения матча | «меньше 2 игроков — стоп» (хардкод) | «меньше `minPlayers` — стоп» |

## Технические изменения (порт 1.12 → 1.16)

- Материалы: `WORKBENCH` → `CRAFTING_TABLE`, `ENCHANTMENT_TABLE` → `ENCHANTING_TABLE`, `MONSTER_EGGS` → `INFESTED_STONE`, `IRON_FENCE` → `IRON_BARS`, `GOLD_PICKAXE` → `GOLDEN_PICKAXE`, `SKULL_ITEM` → `PLAYER_HEAD`.
- Цветные блоки: было `STAINED_GLASS_PANE`/`WOOL` + числовой data (`(short) 5`), стало по имени (`LIME_WOOL`, `BLACK_STAINED_GLASS_PANE`); `fillFreeSlots(inv, short data, ...)` → `fillFreeSlots(inv, String color, ...)`.
- Контракты FIND/MINE/BREAK: сравнение по **числовым ID** (`item.getTypeId()`) → по **имени материала** (`Material.name()`).
- Детект сундучного инвентаря: строка `"container.chest"` → `InventoryType.CHEST`.
- Убран вызов `ItemBuilder.damage()` у вилки (в 1.13+ прочность через Damageable).

## Незавершённые заделы 2021

В 2021 добавлены поля, которые нигде не используются (планы на будущее):

- `NPCsLocations`, `NPCInGameLocations`, `NPCsTasks` — система NPC с задачами (видимо, квестодатели).
- `eventID` геттер/сеттер — доступ к текущему событию извне (не используется).

## Вывод для возрождения

База для новой версии — **логика 2021**: она самая полная и уже «пережила» один порт (1.12 → 1.16), т.е. большая часть версионных ловушек в ней решена. Из 2020 стоит взять одну идею — **одноразовую/ломкую вилку** как осознанное балансовое решение (см. вопросы на согласование).
