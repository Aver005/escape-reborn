# Гайды по Escape

Практическая документация для администратора: как создать арену, наполнить её
контентом, настроить баланс и эксплуатировать на живом сервере.

Геймдизайн и история проекта — в [`docs/`](../README.md) уровнем выше.

---

## С чего начать

**Впервые ставите плагин?** Идите по порядку:

1. [03-arena-create.md](03-arena-create.md) — создайте первую арену
2. [04-arena-content.md](04-arena-content.md) — наполните её контентом
3. [05-arena-tuning.md](05-arena-tuning.md) — настройте баланс
4. [06-operations.md](06-operations.md) — прогоните соло-тест и откройте игрокам

**Нужен конкретный ответ?** Смотрите справочники: [01-commands.md](01-commands.md)
(команды) и [02-configs.md](02-configs.md) (конфиги).

## Пять вещей, которые ломают арены чаще всего

1. **Правки конфигов затираются.** Плагин перезаписывает файлы при выключении и
   по `/escape save`. Правьте при остановленном сервере →
   [02](02-configs.md#золотое-правило-редактирования)
2. **Двойные сундуки не работают.** Рефилл, LOOT-контракты и «Волшебный ключик»
   их не видят. Не ставьте точки вплотную → [03](03-arena-create.md#22-сундуки--обязательно-в-сценарии-a)
3. **Таймеры считаются от остатка времени.** На дефолтах зарплата выдаётся
   один раз за матч → [05](05-arena-tuning.md#как-на-самом-деле-считаются-таймеры)
4. **Окна меню конечны**: 28 товаров и 7 темок на NPC, лишнее не показывается →
   [04](04-arena-content.md#жёсткие-лимиты-вместимости)
5. **`/escape reload` останавливает все идущие матчи** → [06](06-operations.md#ежедневные-операции)
