# Escape — документация по мини-игре

**Escape** — мини-игра для Minecraft-сервера (Spigot-плагин) в жанре **last man standing с тюремной тематикой**: игроки-«заключённые» высаживаются на арену-тюрьму, ломают решётки золотой «Вилкой», собирают золото, выполняют контракты, торгуются с жителями и выживают до финальной битвы. Побеждает последний выживший.

Автор оригинала: **Aver005** (пакет `kiviuly.escape` / `kiviuly.Escape`).

Эта документация составлена по исходникам трёх старых версий из папки `.old/` и служит основой для возрождения игры на современной версии Minecraft.

## Суть игры в трёх абзацах

Игроки заходят в лобби арены через GUI-меню или команду. Пока набирается народ, работает разминка «Избиение в лобби» — урон не наносится, но копится в статистике, и перед стартом объявляют «чемпиона разминки». При наборе минимума игроков — отсчёт 20 секунд и телепорт на арену.

На арене каждый получает **«Вилку...»** (золотая кирка — единственный инструмент побега, ломает железные решётки), **24 золота** (валюта) и **«Личного проводника»** (бумага-навигатор с 4 способностями подсказок). Вокруг — случайно расставленные сундуки с лутом и бумагами-контрактами (6 типов заданий за золото), торговцы-жители с магазинами, рудные жилы (среди которых ~21% — ловушки с чешуйницами). Каждые ~3,5 минуты происходит случайное событие («встаньте под крышу, иначе молния» и т.п.), каждые 10 минут всем капает золото.

Матч длится 30 минут. За 10 минут до конца все игроки подсвечиваются (прятаться больше нельзя), а на нуле таймера всех телепортируют на точки спавна для **финальной битвы**. Смерть — не настоящая: вещи выпадают, игрок становится наблюдателем с шуточным сообщением («Игрок X споткнулся»). Последний выживший — победитель, в чат выводится его статистика: выполненные квесты, убийства, сделки.
