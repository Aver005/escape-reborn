# escape-reborn

Paper-плагин мини-игры **Escape** («побег из тюрьмы», last man standing).
Java 25, Paper 26.1.2, Gradle.

**Единственная инструкция: прочитай [CLAUDE.md](CLAUDE.md) и следуй ему.**
Там порядок работы, команды сборки/деплоя, железные правила и определение
«сделано». База знаний — `.memories/INDEX.md`.

```bash
./gradlew build     # сборка
./gradlew deploy    # сборка + копия в тестовый сервер E:/Servers/escape
```
