# GLOSSARY — термины игры

Last updated: 2026-07-18

- **Вилка** — золотая кирка из стартового набора; одноразовый (fork-uses)
  инструмент для ломания решёток. Тюремный флейвор.
- **Решётка** — IRON_BARS; единственный «строительный» блок, который можно
  ломать в матче; откатывается после игры. «Двери» между зонами карты.
- **Контракт** — задание на бумаге «Задание» из сундука (6 типов:
  KILLS/ACTIVATE/MINE/FIND/BREAK/LOOT); прогресс хранится в самой бумаге.
- **idle** — параметр-цель контракта (имя материала или имя рычага). Название
  исторические, из оригинала.
- **Торговец** — житель-NPC с магазином; «тип торговца» (TraderType) — глобальный
  шаблон, на арене — точки спавна типов.
- **Личный проводник / помощник** — бумага из набора → GUI с 4 подсказками
  координат (сундук/игрок/торговец/локации) под кулдаунами.
- **Отмеченные локации** — именованные рычаги арены; двойная роль: цели
  ACTIVATE-контрактов и точки интереса для проводника.
- **Разминка / «Избиение в лобби»** — урон в лобби обнуляется, но копится;
  перед стартом объявляется топ-3.
- **Зарплата** — периодическая раздача золота всем живым (визуальная молния).
- **Свечение** — за N сек до конца все игроки получают перманентный Glowing.
- **Финальная битва** — на нуле таймера всех телепортируют на финальные спавны
  (или обычные), матч идёт до последнего живого без таймера.
- **Фейковая смерть** — летальный урон отменяется, вещи выпадают, игрок →
  Spectator; ванильной смерти/респавна нет.
- **Маркер** — предмет с PDC-меткой из `add*`-команд; установка блока добавляет
  точку арены (маяк=спавн, сундук, верстак=торговец, рычаг, камень=руда,
  стол зачарований, лодстоун=финальный спавн).
- **Рефилл** — пустой закрытый сундук через 3 мин наполняется заново; стойка
  под сундуком показывает «Пустой»/«Заполнен».
- **Блок возрождения** — личный блок игрока (медь→железо→золото→алмаз→изумруд):
  пока стоит и есть заряды, смерть возвращает на блок. Прокачка через ПКМ-меню,
  переносится, ломается врагами за награду. Молнии выдают позицию.
- **Заряды** — оставшиеся возрождения блока (медь 1, каждая прокачка +2).
- **Молния прозрения** — редкий предмет (громоотвод) из сундуков: молния по
  всем установленным блокам возрождения. Квота на матч: 1 + пары игроков.
- **Аннуляция** — уничтожение всех блоков возрождения при старте финальной
  битвы; дальше смерть окончательна.
- **Оффлайн-страж** — AFK-моб на месте вышедшего живого игрока (90 сек,
  экипировка и ник игрока, не горит днём). Вид = уровень блока возрождения:
  медь — зомби, железо — кадавр, золото — свинозомби, алмаз — утопленник,
  изумруд — зомби-житель. Убит игроком = смерть владельца;
  истёк = дроп вещей. Возврат владельца: при зомби — продолжает; после — только
  через блок возрождения в окне 4 минут от выхода.
- **Снапшот** — полное сохранение состояния игрока перед матчем
  (snapshots/<uuid>.yml), восстанавливается после.
- **Сессия (GameSession)** — один живой матч на арене; арена без сессии = покой.
